# Модель интеллектуального анализатора кардиосигнала

Выпускная квалификационная работа (РУДН, 2025)

## Описание

Гибридная модель анализа кардиосигнала, объединяющая:
- **Систему 5 ОДУ** — гемодинамика сердца (5 камер: LA, LV, Aorta, RA, RV)
- **Reservoir Computing** — аппроксимация нелинейных параметров (Pperi, Vspt)
- **Neural ODE** — интеграция нейросети в систему дифференциальных уравнений
- **Верификация через ЭКГ** — электромеханическое сопряжение

Реальные записи ЭКГ загружаются с
[PhysioNet ECG-ID Database](https://physionet.org/content/ecgiddb/1.0.0/).

## Математическая модель

### Система 5 ОДУ (сохранение массы)

    dV_LA/dt  = Q_pulm_vein - Q_mitral
    dV_LV/dt  = Q_mitral    - Q_aortic
    dV_Ao/dt  = Q_aortic    - Q_systemic
    dV_RA/dt  = Q_vena_cava - Q_tricuspid
    dV_RV/dt  = Q_tricuspid - Q_pulmonary

### Давления (закон Франка-Старлинга)

- Желудочки: P = E(t) * (V - V0) + Pperi
- Предсердия: P = E(t) * (V - V0)
- Аорта: P = (V - V0) / C

### Клапаны (диоды)

Поток разрешён только при положительном градиенте давления:
Q = max((P_in - P_out) / R, 0)

## Верификация через ЭКГ

Электромеханическое сопряжение: ЭКГ задаёт временные метки
для проверки корректности производных dV/dt.

| Событие ЭКГ | Фаза | Ожидание dV_LV/dt |
|---|---|---|
| Зубец P | Предсердная систола | Малый рост (+) |
| QRS | Изоволюметрическое сокращение | Приблизительно 0 |
| ST-сегмент | Выброс | Падение (-) |
| Зубец T | Изоволюметрическое расслабление | Приблизительно 0 |
| Интервал T-P | Наполнение | Быстрый рост (+) |

## Установка

    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt

## Запуск

    python hybrid_ODE_reservoir_with_Physionet.py

Программа:
1. Скачивает реальную запись ЭКГ с PhysioNet
2. Находит P-зубцы, QRS-комплексы, T-зубцы
3. Вычисляет ЧСС и период цикла из RR-интервала
4. Решает систему 5 ОДУ с Reservoir Computing
5. Верифицирует модель: проверяет dV_LV/dt в каждой фазе ЭКГ
6. Выводит 8 графиков и таблицу верификации

## Результаты

Программа генерирует 8 графиков:

| № | График |
|---|--------|
| 1 | Реальное ЭКГ с P, QRS, T пиками |
| 2 | Кардиоциклы + средний референсный |
| 3 | Объёмы 5 камер сердца |
| 4 | Давления (закон Франка-Старлинга) |
| 5 | dV_LV/dt с разметкой фаз ЭКГ |
| 6 | Эластанс E(t) синхронизирован с ЭКГ |
| 7 | Суррогатный ЭКГ vs реальный |
| 8 | Сводная таблица верификации (5 критериев) |
